# GLAB

![](/assets/Capture.C.JPG)

This serves as my final project for **Computer Graphics** course **2017 Fall**.<br/>
I may continue improving it for fun.

## Features
- Deferred rendering 
- PBR metallic / roughness
- Fresnel Schlick
- Microfacet / GGX+Smith 
- Cook-Torrance BRDF 
- Image-based lighting
